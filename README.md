Ayora is a web-based application designed to help users **memorize the Qur’an** in an organized and consistent way. It provides tools for reading, listening, tracking, and reviewing Qur’an memorization progress.

## Features

- **Qur’an Memorization:** Select a surah and ayah, read the Arabic text, view translations, and listen to audio recitation.  
- **Daily Challenge:** Get a new random ayah each day to memorize and mark it as completed.  
- **Progress Tracking:** Save memorization status (new, reviewing, memorized) in the browser.  
- **Responsive Design:** Works on both mobile and desktop devices.  
- **Interactive UI:** Clean and easy-to-use cards, buttons, and visual feedback.  

## Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- [AlQuran Cloud API](https://alquran.cloud/api) for Qur’an text, translations, and audio
